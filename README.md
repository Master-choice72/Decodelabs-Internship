# DecodeLabs Data Analytics Internship — Batch 2026

## About This Repository
This repository documents all projects completed during my 
Data Analytics internship at DecodeLabs. Each project builds 
on the previous one — from raw dirty data to SQL-powered insights.

## Projects Completed

### Project 1 — Data Cleaning & Preparation
- Tool: Microsoft Excel 2010
- Dataset: 1,200 e-commerce orders
- What I did: Fixed 1,000 date errors, 65 floating point 
  errors, 252 blank cells, verified zero duplicates
- File: CLEANED DATA(1).xlsx | Data_Cleaning_Report (2).docx

### Project 2 — Exploratory Data Analysis (EDA)
- Tool: Microsoft Excel 2010
- What I did: Descriptive statistics, IQR outlier detection, 
  5 pivot tables, charts dashboard, 8 KPI cards
- Key Finding: 41.4% of orders cancelled or returned
- File: EDA ANALYSIS.xlsx | Project2_EDA_Report.docx

### Project 3 — SQL Data Analysis
- Tool: SQLite Online (sqliteonline.com)
- What I did: 9 SQL queries using SELECT, WHERE, 
  GROUP BY, ORDER BY, COUNT, SUM, AVG
- Key Finding: Chairs top product, Instagram top channel
- File: PROJECT 3 SQL.xlsx
